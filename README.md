# Scientific Calculator Compiler

## Functions implemented:

- **- expr** 
- **++var**
- **--var**
- **expr + expr**
- **expr - expr**
- **expr * expr**
- **expr / expr**
- **expr % expr**
- **expr ^ epxr**
- **(expr)**
- **var  = expr**

### Comparisons functions implemented:

- **expr1 < expr2**
- **expr1 <= expr2**
- **expr1 > expr2**
- **expr1 >= expr2**
- **expr1 == expr2**
- **expr1 != expr2**

### Boolean functions implemented:

- **!expr**
- **expr1 && expr2**
- **expr1 || expr2**

### Precedence rules:

- **operator ||**
- **operator &&**
- **operator !**
- **assignment operators**
- **operator +, -**
- **operator ***
- **operator ^**

## Installation

### Prerequisites

- GCC / CC
- Bison / Yacc
- Flex / Lex
- make

### Commands

```shell
make
```
